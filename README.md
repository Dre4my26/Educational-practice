## Little description
_parser.py_ - parses comments from website "**habr.com**" from articles dedicated to Machine Learning and collects it to articles.csv

_articles.csv_ - data file which consists of: 
- article_id 
- article_title 
- article_link 
- number_of_comments 
- comment 
- commentator 
- theme
- comments_under_comment
- date
